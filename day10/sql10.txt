primary key ---> unique + not null
 Uniquely identifies each row in a table.

create table std (id int primary key , name varchar(20) not null,
company varchar(20) default "vector");

insert into std (id,name) values (1,'john');//inserted
insert into std (id,name) values (1,'john');//error
insert into std (id,name) values (null,'john');//error

every table must contain one primary key

auto_increment

means adds a one number for every insertion

foreign key
Ensures referential integrity by linking two tables.
Note: The referenced table should have a PRIMARY KEY.
Foreign Key column can have duplicates and NULL values.
Parent table: The table with a PRIMARY KEY. and foreign key 
Child table: The table referencing the PRIMARY KEY.

create table emp (deptno  int primary key, dname varchar(20));

create table dept (empno int primary key auto_increment,
 name varchar(20) not null,deptno int,
foreign key(deptno) references emp(deptno));

we cannot delete child table without deleting the parent table

desc emp;//child
desc dept;//parent
